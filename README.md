# Shadow Chat for Desktop

The desktop app for **Shadow Chat** — a private, end-to-end encrypted messenger.

Built with Electron. The Windows installer is produced in CI and published under
the repository's releases.

**Download (Windows):**
https://github.com/myshadowchat/shadow-chat-desktop/releases/download/windows/Shadow-Chat-Setup.exe

## Development

```bash
npm install
npm start          # run locally
npm run dist       # build the Windows installer
```
